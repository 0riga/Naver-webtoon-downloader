## Что это такое?

Это программа, которая **автоматически скачивает комиксы** с корейского сайта Naver Comic.

**Что она умеет:**
- ✅ Скачать все главы тайтла
- ✅ Скачать с конкретной главы до конца
- ✅ Скачать только одну главу
- ✅ Работает с любыми ссылками на тайтл

## Установка Python (пошагово)

### Шаг 1: Скачайте Python

1. Откройте браузер
2. Перейдите на сайт: **https://www.python.org/downloads/**
3. Нажмите большую кнопку **"Download Python"** (версия 3.11 или новее)

### Шаг 2: Установите Python

1. Запустите скачанный файл (например, `python-3.11.0.exe`)
2. ⚠️ **ВАЖНО:** Поставьте галочку ✅ **"Add Python to PATH"** (внизу окна!)
3. Нажмите **"Install Now"**
4. Дождитесь окончания установки
5. Нажмите **"Close"**

### Шаг 3: Проверьте установку

1. Нажмите **Win + R** на клавиатуре
2. Введите `cmd` и нажмите **Enter**
3. В чёрном окне введите:
   ```
   python --version
   ```
4. Должно появиться: `Python 3.11.X` (или другая версия)

**Если увидели версию Python** → ✅ Установка успешна!

**Если ошибка** → Переустановите Python с галочкой "Add Python to PATH"

## Установка зависимостей

### Шаг 1: Откройте командную строку

1. Нажмите **Win + R**
2. Введите `cmd`
3. Нажмите **Enter**

### Шаг 2: Перейдите в папку со скриптом

Введите команду (измените путь на свой):
```cmd
cd C:\Users\ВАШЕ_ИМЯ\webtoon_parser
```

**Как узнать свой путь:**
1. Откройте папку со скриптом в Проводнике
2. Кликните в адресной строке сверху
3. Скопируйте путь
4. Вставьте после `cd ` в командной строке

### Шаг 3: Установите библиотеки

Введите команду:
```cmd
pip install -r requirements.txt
```

Дождитесь окончания установки. Должно появиться:
```
Successfully installed selenium-... requests-... beautifulsoup4-...
```

### Шаг 4: Установите ChromeDriver

Скрипт использует Google Chrome для открытия страниц.

**Автоматическая установка:**
```cmd
pip install webdriver-manager
```

**Вручную (если нужно):**
1. Скачайте ChromeDriver: https://chromedriver.chromium.org/downloads
2. Выберите версию под вашу Chrome
3. Распакуйте `chromedriver.exe` в папку со скриптом

## Как использовать

### Базовая команда

```cmd
python download_all.py -u "ССЫЛКА" [ОПЦИИ]
```

### Опции

| Опция | Что делает | Пример |
|-------|------------|--------|
| `-u "URL"` | **Обязательно!** Ссылка на тайтл | `-u "https://..."` |
| `-n "НАЗВАНИЕ"` | Начать с главы по названию | `-n "어린이집"` |
| `-s НОМЕР` | Начать с главы по номеру | `-s 15` |
| `-1` | Скачать **только одну** главу | `-1` |
| `-o "ПАПКА"` | Папка для сохранения | `-o "downloads"` |

## Примеры команд

### 1️⃣ Скачать ВСЕ главы тайтла

```cmd
python download_all.py -u "https://comic.naver.com/webtoon/list?titleId=812354"
```

**Что сделает:**
- Скачает все главы (218 штук)
- Сохранит в папку `downloads/`

---

### 2️⃣ Скачать с 15 главы до конца

```cmd
python download_all.py -u "https://comic.naver.com/webtoon/list?titleId=839544" -s 15
```

**Что сделает:**
- Скачает главы 15, 16, 17, ... до последней
- Пропустит главы 1-14

---

### 3️⃣ Скачать ТОЛЬКО ОДНУ главу (по номеру)

```cmd
python download_all.py -u "https://comic.naver.com/webtoon/list?titleId=812354" -s 79 -1
```

**Что сделает:**
- Скачает только главу №79
- Остановится после неё

---

### 4️⃣ Скачать ТОЛЬКО ОДНУ главу (по названию)

```cmd
python download_all.py -u "https://comic.naver.com/webtoon/list?titleId=812354" -n "어린이집" -1
```

**Что сделает:**
- Найдёт главу со словом "어린이집" в названии
- Скачает только её

---

### 5️⃣ Скачать с конкретной главы по названию

```cmd
python download_all.py -u "https://comic.naver.com/webtoon/list?titleId=812354" -n "드레스투어"
```

**Что сделает:**
- Найдёт главу со словом "드레스투어"
- Скачает её и все последующие

---

### 6️⃣ Скачать в свою папку

```cmd
python download_all.py -u "https://comic.naver.com/webtoon/list?titleId=812354" -o "Мои_комиксы"
```

**Что сделает:**
- Создаст папку `Мои_комиксы/`
- Сохранит все главы туда

---

### 7️⃣ Скачать по ссылке на главу

```cmd
python download_all.py -u "https://comic.naver.com/webtoon/detail?titleId=812354&no=162&week=thu" -1
```

**Что сделает:**
- Автоматически определит главу №162
- Скачает только эту главу

## Частые проблемы

### ❌ Ошибка: `'python' is not recognized`

**Решение:**
1. Python не добавлен в PATH
2. Переустановите Python
3. ✅ Поставьте галочку **"Add Python to PATH"** при установке
4. Перезапустите командную строку

---

### ❌ Ошибка: `No module named 'selenium'`

**Решение:**
```cmd
pip install -r requirements.txt
```

---

### ❌ Ошибка: `chromedriver not found`

**Решение 1 (автоматически):**
```cmd
pip install webdriver-manager
```

**Решение 2 (вручную):**
1. Скачайте ChromeDriver: https://chromedriver.chromium.org/downloads
2. Выберите версию под ваш Chrome
3. Распакуйте `chromedriver.exe` в папку со скриптом

---

### ❌ Ошибка: `HTTP Timeout` / `Read timed out`

**Причина:** Медленный интернет или сервер не отвечает

**Решение:**
1. Подождите немного и запустите снова
2. Скрипт автоматически повторит неудачные загрузки
3. Проверьте интернет-соединение

---

### ❌ Ошибка: `Chapter not found`

**Причина:** Глава с таким названием/номером не найдена

**Решение:**
1. Проверьте номер главы на сайте
2. Используйте часть названия (не полное)
3. Пример: `-n "42화"` вместо `-n "시즌2 42화 첫 영화관"`

---

### ❌ Скрипт скачал не все главы

**Причина:** Некоторые изображения не загрузились

**Решение:**
1. Скрипт автоматически пытается загрузить неудачные главы
2. Запустите команду снова — он продолжит с места остановки
3. Проверьте папку — некоторые главы могли загрузиться частично
